Viducation

A two-sided, video-streaming learning platform that features credit card payment capabilities for premium courses, user role management for teachers and students, complex user interfaces, and advanced database relationships.

Technology

Viducation uses Bootstrap3 for a mobile-first layout, with JavaScript and jQuery for easy page manipulation. Stripe is used for secure credit card payments, Devise for user security, ImageMagick for photo auto-resizing, and Carrierwave for image/video uploads. On the backend a Rails app lives on a Heroku dyno all created within a Vagrant environment.
